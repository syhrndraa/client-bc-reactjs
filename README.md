## Run Locally

Clone the project

```bash
  git clone https://github.com/syhrndraa/client-bc-reactjs
```

Go to the project directory

```bash
  cd client-bc-reactjs
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
